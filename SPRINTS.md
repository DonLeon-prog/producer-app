1. Открываешь Claude Code в терминале (`claude` в папке проекта)
2. Берёшь задачу из спринта

## Кто наши пользователи

Приложение работает в двух режимах — пользователь выбирает кто он при онбординге:

**Блогер** — снимает короткие видео, хочет вырасти в Instagram / TikTok / YouTube Shorts.
**Музыкант** — записывает треки, хочет продвинуться на стримингах и набрать аудиторию.

Все AI-функции, контент-планы и советы разные для каждого типа. В базе данных это одно поле `user_type`.

### Задача 2.5 — API для профиля (оба типа)

**Вставь в Claude Code:**
```
В backend/app/api/profile.py:

GET /api/profile
- Возвращает профиль пользователя

- Возвращает {"profile": {...}, "user_type": "...", "analysis": "текст анализа"}

В backend/app/core/limiter.py:
- check_limit(user_id, plan) → bool
  Free: 5 запросов в день
  Pro и Producer: безлимит
- increment(user_id) — увеличивает счётчик, TTL до конца дня
```

## Важные команды

```bash
# Запустить локально
docker-compose up -d
cd backend && uvicorn app.main:app --reload
cd frontend && npm run dev
```
